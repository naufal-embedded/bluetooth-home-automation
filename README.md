# bluetooth-home-automation
# Task 2: Home Automation with Bluetooth

## Objective
To control home appliances using Bluetooth communication.

## Components Used
- ESP32
- Relay Module
- Bluetooth (ESP32 inbuilt)
- Mobile Bluetooth Terminal App

## Working
The ESP32 receives commands via Bluetooth from a mobile phone.
Based on the received command, relays are switched ON or OFF.

## Output
Appliances are controlled wirelessly using Bluetooth.
